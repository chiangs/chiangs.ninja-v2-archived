Presidents Web

A collaborative project using servlets and Java Server Pages (JSP)

This project utilized servlets and data access objects (DAO) to query and serve information from a CSV translated to html via JSPs on presidents of the United States. The other challenge in logic was cycling through the end back to the beginning of the information to provide an uninterrupted user experience.

Inventory Management Web-app

An inventory management web-app using Java and the Spring MVC framework with CRUD functionality.

This is my first project utilizing Springs Model and View (MVC) that allows the user to manage inventory. Although this web-app implements session control, the database does not persist and reverts back to the original state of pre-loaded inventory when the server is restarted. This web-app is also largely mobile friendly and works on mobile devices.

Human Resources Management Web-app

An employee management web-app using Java and the Spring MVC framework with CRUD functionality.

This is my first project incorporating a MySQL database and implementing session control. The web-app is also mobile responsive and data can be viewed/manipulated with most popular mobile devices. The database was created with MySQL Workbench and managed via command-line.

Travel Review Social Web-app

A web-app for users who travel and wish to share reviews and ratings of places all over the world.

This is a collaborative project developed with the Java JPA API, Spring MVC framework, and MySQL database technologies with CRUD functionality. This web-app implements session control, Object-Relational Mapping (ORM), Form Validation, and my first attempt at a login/logout security feature.

JUnit tests were written to ensure proper mapping of the entities before moving from JPA to SpringMVC. Then tests were written for the DAO methods to ensure proper CRUD functionality and session persistence of a user.

Consumption Tracker RESTful Web-app and API (refactored with AngularJS)

A web-app for a F&B manager who wishes to check on consumption and re-order.

This is a RESTful web-app developed on a stack consisting of the Java JPA API, the Spring framework JavaScript/jQuery and a MySQL database. The user is able to manipulate data from the database and create, update, delete, and review overall tracking statistics which are updated asynchronously based on user input. A fuzzy search function via AngularJS is also implemented.

RESTful Web-app to find and rent camera equipment

Similar to AirBnB for camera equipment.

This is a RESTful web-app consisting of the Java JPA API, the Spring framework and a MySQL database. The data is consumed and displayed using AngularJS. User are able to register as a renter or lister, create inventory, search for equipment and book reservations. The app also uses several modules from the Google Maps API to dynamically display locations of stores.
